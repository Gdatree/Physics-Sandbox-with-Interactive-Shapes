# Physics-Sandbox-with-Interactive-Shapes
Interactive 2D physics sandbox using Pygame &amp; PyMunk. Spawn cubes, balls, a super-bouncy pink ball, and triangles. Drag, rotate, and collide shapes with realistic physics. Colors distinguish objects: blue cube, green ball, pink bouncy ball, yellow triangle. Experiment with collisions, elasticity, and movement.
